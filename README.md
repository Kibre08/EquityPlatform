# Equity Platform

## Project Description
A full-stack Equity Platform built using the MERN stack.

## Setup Instructions
1.  Clone the repo: `git clone https://github.com/Kibre08/EquityPlatform`
2.  Install dependencies: `npm install`
3.  Run the server: `npm run dev`

## Project Status & Scope
> **Note on Accessibility:** As outlined in the project documentation, this platform is architected as a **Globally Accessible Web Application**. 
>
> *Current Deployment Status:* For the purpose of submission, testing, and data security, the application is currently configured to run in a **Local Sandbox Environment**. This demonstrates the full functionality of the system without the security risks of an immediate public launch. The architecture is fully cloud-ready (as detailed in `DEPLOYMENT.md`).
